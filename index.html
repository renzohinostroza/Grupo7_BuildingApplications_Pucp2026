Descripción General del Proyecto

Este proyecto implementa una solución automatizada para una planta cervecera industrial, permitiendo registrar, monitorear y analizar el cumplimiento del Plan Maestro de Producción (PPA).

La arquitectura propuesta está basada en la nube y diseñada para consolidar información operativa en tiempo real, optimizar su procesamiento mediante flujos ETL y proporcionar acceso seguro y flexible mediante aplicaciones low-code y herramientas modernas de inteligencia de negocios.

Arquitectura del Proyecto

Data Layer

La base de datos fue diseñada para gestionar eficientemente el cumplimiento del Plan de Producción integrando planificación, producción real y variables de afectación.

tb_plan_de_produccion

Registro de planificación operativa.

tb_produccion_real

Resultados reales y cálculo del indicador PPA.

tb_variables_presentadas

Registro de pérdidas y desviaciones.

tb_variables_de_afectacion

Catálogo de causas raíz operativas.

Application Layer

ETL Layer

El sistema emplea un Dataflow automatizado utilizando Microsoft Fabric, Azure SQL Database y notebooks en Python.

DWH Layer

La capa DWH consolida información histórica para análisis avanzado, reporting empresarial y construcción de indicadores operativos.

Presentation Layer - Power BI

Dashboards interactivos desarrollados en Power BI para responder preguntas de negocio y monitorear KPIs operativos.
